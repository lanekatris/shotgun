Shotgun Rules

  1. All persons partaking in said carriage ride must be outside
  2. All persons should be able to see the carriage
  3. "Shotgun" shall not be callest until the last person's foot is outside the cottage
  4. If "Shotgun" is callest at the same time by multiple people, a coin flip or rock paper scissors must be used to break the tie
  5. "Shotgun" duration is only applicable one way to the destination. "Shotgun" can be called during the return trip per the rules above
